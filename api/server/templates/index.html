Welcome to the SAT-Sample API

Description

This API provides a simple CRUD interface for managing SAT questions. Each question has a question, solution, correct answer, options, steps, an image URL and other optional key value pairs. The source code for this API can be found on GitHub

Endpoints Download postman collection

  • Create: POST /api/question
  • Read: GET /api/question/{id}
  • Update: PUT /api/question/{id}
  • Delete: DELETE /api/question/{id}
  • List: GET /api/questions
